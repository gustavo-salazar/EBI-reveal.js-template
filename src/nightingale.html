What is Nightingale?

Nightingale is a library of **re-usable** data visualisation Web Components, which can be used to display **protein sequence features**, variants, interaction data, 3D structure, etc. These components are flexible, allowing you to easily view **multiple data sources** (UniProt, InterPro, your own resource, etc) within the same context.

UniProt Protvista

Graphical representation of protein sequence features in the **UniProt** Knowledgebase
